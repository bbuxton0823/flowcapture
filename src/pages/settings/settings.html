YouTube (Publish Videos)

Not authorized

One-click upload of generated MP4s to YouTube as unlisted videos. The embed link is then pasted into Yardi Aspire.

How to set up YouTube credentials (one-time, ~5 min)
  1. Go to Google Cloud Console → Create Project → "FlowCapture-HACSM"
  2. Enable the YouTube Data API v3 (APIs & Services → Library)
  3. OAuth consent screen → External → fill required fields → add scope youtube.upload
  4. Credentials → Create OAuth client ID → Web application
  5. Authorized redirect URI: loading...
  6. Paste the Client ID and Client Secret below
  7. Click "Authorize YouTube" — sign in with the HACSM YouTube channel owner's Google account

Vimeo (Publish Videos)

Not configured

Alternative to YouTube — useful if HACSM IT prefers Vimeo. Generates the same embed link for Aspire.

How to get a Vimeo access token (one-time, ~3 min)
  1. Go to developer.vimeo.com/apps → Create App
  2. App name: "FlowCapture-HACSM"
  3. Click Generate Personal Access Token
  4. Check scopes: upload, edit, delete, video_files, private, public
  5. Copy the token and paste below
  6. Note: requires Vimeo Plus or higher to upload videos
