FlowCapture

Record any browser workflow as a narrated video SOP — one click, no screen recorder needed.

⬇ Download & Install

Free · Open source · Chrome extension · No account required

What it does

Capture, narrate, and publish standard operating procedures without leaving the browser.

📸

Auto-capture

Click through any workflow and FlowCapture screenshots every step automatically — with URL, element, and timestamp.

🎬

One-click video

Turns captured steps into a 1080p MP4 with intro/outro, AI narration (ElevenLabs or browser TTS), and caption tracks.

📄

PDF export

Generates a polished PDF SOP with cover page, table of contents, role badges, and approval stamps.

🎙️

AI Narration

Optional ElevenLabs integration for professional voice-over. Or use built-in browser TTS — no API key needed to get started.

☁️

Publish anywhere

Upload directly to YouTube or Vimeo (unlisted), copy the embed URL, and paste into Yardi Aspire or any LMS.

🔒

Local-first

All screenshots and recordings stay in your browser. No backend, no cloud, no account required.

Install in 60 seconds

Three short steps. No app store, no sign-in.

Step 1 Download

⬇ Download FlowCapture.zip

Or clone with git:

git clone https://github.com/bbuxton0823/flowcapture.git

Step 2 Load in Chrome

  1. Open Chrome and go to chrome://extensions
  2. Turn on Developer mode (toggle in the top-right)
  3. Click "Load unpacked"
  4. Select the flowcapture folder you downloaded/cloned
  5. The FlowCapture icon appears in your toolbar — you're ready!

Step 3 Optional: AI Narration

Set up ElevenLabs for professional voice-over (optional)
  1. Get a free ElevenLabs API key at elevenlabs.io
  2. Open FlowCapture → Settings → ElevenLabs
  3. Paste your key and click Load Voices

Skip this step to use the built-in browser TTS — no API key needed.

Quick start

From install to published video in three clicks.

1

Click the FlowCapture icon → "Start Capturing" → browse through your workflow.

2

Click "Stop" → review and edit steps in the Editor.

3

Click "Generate Video" or "Export PDF" — done!
